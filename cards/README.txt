Модуль печати карточек для биллинговой системы NoDeny.

=================Установка=================

1. Копируем файл operations.patch на сервер в web директорию NoDeny.
/usr/local/nodeny/web (что бы он оказался рядом с operations.pl). Делаем 
резервную копию operations.pl.

2. Выполняем патч для файла operations.pl следующей командой:
patch -i operations.patch

3. Если необходимо - меняем владельца на www для файла operations.pl
chown www:www operations.pl

4. Если нужно деньги привести к западному типу разделителя целой и дробной части
нужно закоментировать строку $p->{money} =~ tr/./,/; (поставить # вначале строки).

5. Если необходимо работать с номиналами более 9999.99, то снимаем ограничение 
в SQL биллинга следующим запросом:
ALTER TABLE `cards` CHANGE `money` `money` FLOAT( 11, 2 ) NOT NULL DEFAULT '0.00'

6. Браузером заходим в админку NoDeny, меню "Операции". Если карточки не 
сгенерированы, проходим в меню "Карточки пополнения счёта" и генерируем их.
Если карточки уже имеются в меню "Операции" ищем справа сверху кнопку
"Операции ->".

7. Активируем карточки.

8. Появляется столбец "Экспорт в CSV" . Нажимаем кнопку "Сохранить". Файл с csv
сохранится под названием nocards.txt.

ДАЛЕЕ ИНФОРМАЦИЯ БОЛЬШЕ ДЛЯ ТИПОГРАФИИ
9. При распечатывании в Windows создаём каталог C:\nocards\. Копируем туда файл
nocards.mdb и nocards.txt. Открываем базу nocards.mdb и переходим в отчёты. 
Кликаем по лицевой стороне. Печатаем. Кликаем по задней стороне. Печатаем.

10. Чтобы настроить используемые в Windows числовые и языковые стандарты, необходимо 
выполнить следующую последовательность команд: 
Пуск->Панель управления->Часы, язык и регион->Язык и региональные стандарты.
Разделитель целой и дробной части должен быть "," (запятая).
